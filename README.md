# Roshan Docs

A personal documentation website built with HTML, CSS, and JavaScript.

**Live:** [roshan-mallick.github.io/docs](https://roshan-mallick.github.io/docs/)

## What's Inside

- **Git** — Install, configure, branching, merging, rebasing, GPG signing
- **GitHub** — CLI, auth, repos, forking, branching, pull requests, open source
- **HTML** — Document structure, elements & tags, forms
- **CSS** — Selectors, box model
- **JavaScript** — Variables, functions, arrays, objects, DOM, async/await, ES6 modules
- **C** — Hello world, variables, conditionals, loops, arrays, functions, pointers, strings
- **Linux** — Malware scanning, firewalls, AppArmor, file integrity, hardening, security checklist

## Features

- Sidebar navigation with collapsible categories
- Global search via `Ctrl+K`
- Reading progress bar
- Copy-to-clipboard on code blocks
- Responsive — works on mobile, tablet, and desktop
- Light mode only

## Run Locally

Just open `index.html` in your browser. No build tools needed.
